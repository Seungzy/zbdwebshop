# ZBDWebShop

ZBDWebShop is a simple web application built with **Vue.js**. This project was created as my first experience working with the Vue framework and served as an introduction to building modern front-end web applications using a component-based architecture.

## About the Project

This project was developed during my **Second Year, Second Term** for the course:

**IT SPECIALIZATION 3 – ADVANCE WEB DESIGN**

The objective of the course was to learn a modern front-end framework, and I chose **Vue.js** for this project. It was my first time working with Vue, allowing me to explore its component-based architecture, routing, and overall development workflow.

Since the focus of the course was learning the framework itself, the application remains intentionally simple.

## Features

- Built using Vue.js
- Simple e-commerce-style interface
- Navigation between pages
- Login and Sign Up interface
- Responsive user interface
- Mobile First approach

## Important Notes

- **No database is connected.**
- User credentials entered during **Sign Up** or **Login** are **not stored** or authenticated.
- These pages were included only to demonstrate the front-end functionality because our professor did not require a database implementation for this project.

## Purpose

The purpose of this project was to demonstrate my understanding of the Vue.js framework rather than build a production-ready web application. As my first Vue project, it shows that I was able to set up a Vue application, understand its project structure, build reusable components, and create a working front-end interface.

## Technologies Used

- CSS
